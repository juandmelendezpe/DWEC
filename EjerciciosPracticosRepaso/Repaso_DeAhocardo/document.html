Documentación del Juego del Ahorcado

Estructura del Código

El código implementa un juego del ahorcado utilizando JavaScript. Se divide en variables principales, funciones para la lógica del juego y eventos para la interacción con el usuario.

Variables y Objetos

  • palabras: Array de palabras disponibles para el juego.
  • palabraSeleccionada: Palabra seleccionada aleatoriamente.
  • palabraOculta: Array que representa la palabra oculta con guiones bajos.
  • intentos: Número de intentos restantes.
  • elementos: Referencias a elementos del DOM.
  • contexto: Contexto del lienzo para dibujar el ahorcado.

Funciones

  • seleccionarPalabra:

    Este método selecciona aleatoriamente una palabra del array palabras. Utiliza Math.random() para generar un índice aleatorio y devolver la palabra correspondiente. Es crucial para determinar la palabra a adivinar al inicio del juego.

  • inicializarPalabraOculta:

    Crea un array de guiones bajos (_) cuya longitud coincide con la palabra seleccionada. Utiliza String.repeat() y String.split() para inicializar la representación oculta de la palabra que el jugador irá descubriendo.

  • actualizarVista:

    Actualiza dinámicamente el contenido del DOM para reflejar el estado actual de la palabra oculta. Convierte el array en una cadena con espacios para hacerlo más legible para el jugador.

  • procesarIntento:

    Procesa el intento del jugador verificando si la letra ingresada está presente en la palabra seleccionada. Si está, reemplaza los guiones bajos con la letra correspondiente en las posiciones correctas. Si no, decrementa los intentos y llama a la función dibujarAhorcado para agregar una parte al dibujo del ahorcado.

  • verificarEstado:

    Evalúa si el juego ha terminado. Si no hay más guiones bajos en la palabra oculta, el jugador ha ganado. Si los intentos se reducen a cero, el jugador pierde. Devuelve un mensaje descriptivo y el estado actual del juego.

  • dibujarAhorcado:

    Dibuja el ahorcado en el lienzo HTML5 en función del número de intentos restantes. Utiliza un array de funciones anónimas para agregar progresivamente las partes del dibujo (base, poste, cuerda, cabeza, cuerpo, etc.). Cada parte se dibuja según el índice correspondiente a los intentos restantes.

Eventos

El botón "Comprobar" permite al jugador ingresar una letra para verificar si está en la palabra seleccionada. Se gestionan los intentos, se actualiza la vista y se verifica el estado del juego.
