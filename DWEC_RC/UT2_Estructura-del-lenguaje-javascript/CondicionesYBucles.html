Condicionales y Bucles

Condicionales

Las estructuras condicionales nos permiten ejecutar diferentes bloques de código en función de si una condición es verdadera o falsa.

If...Else

if (condición) {
    // Código a ejecutar si la condición es verdadera
} else {
    // Código a ejecutar si la condición es falsa
}

Switch

        switch (expresión) {
    case valor1:
        // Código a ejecutar si expresión es igual a
        break;
    case valor2:
        // Código a ejecutar si expresión es igual a valor2
        break;
    default:
            
        // Código a ejecutar si expresión no coincide con ningún caso
        break;
}

Bucles

Los bucles nos permiten repetir un bloque de código mientras una condición sea verdadera o durante un número determinado de iteraciones.

For

for (inicialización; condición; incremento) {
    // Código a repetir
}

While

while (condición) {
    // Código a repetir mientras la condición sea verdadera
}

Do...While

do {
    // Código a repetir al menos una vez
} while (condición);

Break

La instrucción break se utiliza para salir de un bucle o una estructura switch antes de que haya terminado su ejecución normal.

Continue

La instrucción continue se utiliza para saltar a la siguiente iteración de un bucle, omitiendo el código restante en la iteración actual.
